NgxBlade
NgxBlade is a Angular component to create blade views similar to those found in the Microsoft Azure Portal.
  • Click on ⊞ button to expand this blade to the full width of the page
  • When expanded, Click on ⊟ button to minimize back to the default/specified width
  • To close the blade, use the × button at the top right corner
Use the ngxBladeHeader and ngxBladeContent inside the ngx-blade selector to specify the header and body of the blade.
